# REST API TMDb

## Guia de Instalação

Para instalar a aplicação deve ser executados os seguintes comandos no terminal, no diretório desejado.
	
	git clone https://github.com/davidaugusto89/rest-api-tmdb rest-api-tmdb

	cd rest-api-tmdb

	composer install

	cp .env.example .env

	php artisan key:generate

	php artisan serve
